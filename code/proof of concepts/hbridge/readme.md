# H-Bridge proof of concept
1) Download de File
2) Upload de code naar de Microcontroller
3) - Beide motoren versnellen vooruit tot hun maximum
   - Bocht naar rechts vooruit
   - Bocht naar links vooruit
   - Beide motoren vertragen achteruit tot 0
   - Beide motoren versnellen van 0 tot hun maximum
   - Bocht naar rechts achteruit
   - Bocht naar links achteruit
   - Beide motoren vertragen achteruit tot 0
   - Beide motoren stoppen
